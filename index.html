The header does not have any positioning applied, therefore assumes the default- which is static
The nav bar is in a fixed position which means it stays in the same place even when the page is scrolled
This article is positioned relatively which causes a relatively positioned element to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
The aside is in an absolute position which means it's positioned relative to the nearest positioned ancestor. If an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with the page scrolling.
